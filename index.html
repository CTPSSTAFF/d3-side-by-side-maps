Side-by-Side D3 MPO Town Maps


Data source: TopoJSON

  1. Spatial data loaded from TopoSON file.
    • Derived from a clipped version of MassGIS TOWNSURVEY_POLYM layer.
    • GeoJSON first generated in ArcMap using Calvin Metcalf's esri-to-open tool.
    • GeoJSON then converted to TopoJSON using geojson.io web service.
    • TopoJSON manipulated using Mike Bostock's topojson.js library.
  2. Maps are symbolized by FOURCOLOR attribute in TopoJSON file, using 2 different color palettes.
  3. Notice how much faster it loads than the version using a GeoJSON data source - the TopoJSON is 1/10 the size of the GeoJSON!
  4. Includes simple D3 on-hover "title" tooltip for each town.
  5. Pan and zoom (from 0.2x to 5x) synchronized between the 2 maps.
